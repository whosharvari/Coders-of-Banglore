# Coders of Bangalore

## Overview

This project parses raw Instagram-style data from a text file, converts it into structured JSON, and performs basic analysis to extract insights.

## Features

* Convert unstructured text → structured data
* Handle `K` and `M` number formats
* Export data to JSON
* Perform basic analytics

## Analysis

* User with maximum posts → `startuphub_blr`
* User with maximum followers → `_anujsinghal`
* User following the most → `bangalore_tech_bro`
* Total categories → `34`

## Tech Stack

* Python
* JSON
* Jupyter Notebook

## Files

* `data.txt` → Raw data
* `data.json` → Processed data
*  Notebook → Analysis
